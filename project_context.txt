SECTION 1: PROJECT OVERVIEW
----------------------------------------------------------------

WHAT THE APP DOES:
A step-by-step web form where a user selects a vehicle
(year → make → model) and answers a few questions to look up
the correct key fob information from a Supabase database.

WHO IS BUILDING IT:
A beginner in web development. All code should include clear
explanations of what it is doing. Avoid jargon without
explaining it first. Changes should be explained line by line
rather than delivering full rewrites — the builder wants to
learn what each change does as they make it.

TECH STACK:
- React (created with npx create-react-app)
- Supabase (cloud database, connected via JavaScript client)
- All code lives in a single App.js file for simplicity
- Plain CSS in App.css (chosen for simplicity and being the
  foundational standard — best for a beginner)

SECTION 5: KEY CODE PATTERNS USED
----------------------------------------------------------------

PAGINATION:
The database has more rows than Supabase returns in one request.
All fetch functions loop using .range() to get every page of
results until there are no more rows left.

CASCADING FILTERS:
Each dropdown is filtered by all previous selections so users
only ever see options that actually exist in the database.

AUTO-SELECT:
If a step only has one possible answer (e.g. only one fob button
count exists for a vehicle), the app skips that step and selects
it automatically without asking the user.

NUMBER CONVERSION:
Year values from dropdowns come back as strings (e.g. "2021").
The database stores years as numbers (e.g. 2021).
Every Supabase query wraps year in Number() to convert it:
  .eq('year', Number(year))

CSS CLASS SYSTEM:
Styles are applied using className= in React (not class= as in
plain HTML). Classes are defined in App.css and imported at the
top of App.js with: import './App.css';

FLEXBOX LAYOUT:
The app uses CSS Flexbox for layout. Key properties used:
  display: flex          — turns on Flexbox
  flex-direction: column — stacks children top-to-bottom
  align-items: center    — centers children horizontally
  gap: 16px              — adds space between children automatically

REFS AND SCROLLING:
useRef() is used to label specific elements on the page.
A scrollTo() helper function smoothly scrolls to those elements.
setTimeout() is used to delay the scroll slightly so React has
time to render the next element before trying to scroll to it.
The scrollTo function accepts a custom delay:
  scrollTo(ref)        — uses default 300ms delay
  scrollTo(ref, 500)   — uses 500ms delay (for slower steps)
